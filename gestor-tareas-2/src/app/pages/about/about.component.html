Sobre esta aplicación

Esta es una aplicación de gestión de tareas desarrollada con Angular. Permite crear, completar y eliminar tareas de manera sencilla y eficiente.

La interfaz está diseñada para ser clara y fácil de usar, con colores suaves y elementos interactivos intuitivos.

Funciones principales:

  • Crear nuevas tareas con título y descripción.
  • Marcar tareas como completadas.
  • Eliminar tareas existentes.
  • Visualización de la lista de tareas en tiempo real.

Esta aplicación es un ejemplo de cómo utilizar Angular Components, servicios y enlace de datos para crear aplicaciones web modernas.
